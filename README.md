# Carhouse
# Live Link : https://carhouse-a8546.web.app/
## Features
- Hamburger menu
- Responsive
- Carousel
- Accordion
- Update Items
- Add Items
- Manage Items
- About Me
- Blogs
- Signin with email and password
- Signin with Google

## Technologies
- React Bootstrap
- React Router
- Font Awesome
- Firebase
- Toastify
- MongoDB
- Node


This website is all about managing a Car warehouse. You can add, update and delete stock list of Car.
